# **App Name**: PICTIAP

## Core Features:

- User Authentication: Firebase authentication for student and teacher accounts.
- Teacher Dashboard: Dashboard to view student performance and manage quizzes.
- Student Dashboard: Dashboard to view available quizzes and performance.
- Quiz Creation: Teachers can create, edit, and publish quizzes.
- Quiz Taking: Students can take quizzes and see their scores.
- Performance Tracking: Track student performance on quizzes and display results.
- Quiz Recommendation Tool: AI-powered feature that analyzes student performance and recommends relevant quizzes for improvement; LLM decides when and if to suggest quizzes based on a reasoning tool.

## Style Guidelines:

- Primary color: Dark green (#386641) for a focused learning environment.
- Background color: Light green (#E5EBE1), a very low saturation tint of the primary color to promote focus and minimize eye strain.
- Accent color: A yellow-green (#84A98C) to highlight important information.
- Body font: 'PT Sans' a versatile sans-serif font, suitable for body text.
- Headline font: 'Space Grotesk', a computerized, techy sans-serif font; since the body text is well-suited to longer passages, this choice gives a modern style.
- Simple, clear icons for navigation and quiz elements.
- Clean, organized layout for easy navigation and information access.
- Subtle animations for feedback and user engagement.
